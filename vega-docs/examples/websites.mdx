# Building Websites with Vega

Vega's **WebAgent** is purpose-built for generating clean, responsive, production-quality websites. From a single prompt, it produces complete HTML, CSS, and JavaScript files — ready to open in any browser.

---

## How It Works

When you describe a website, the WebAgent:

1. **Parses your design intent** — color schemes, layout preferences, content sections
2. **Plans the file structure** — typically `index.html`, `style.css`, `script.js`
3. **Generates each file** — with semantic HTML, modern CSS (Grid/Flexbox), and vanilla JS
4. **Writes to your directory** — files land in a named subfolder in your working directory

## Example 1 — SaaS Landing Page

**Prompt:**
```
Build a landing page for a SaaS product called "Notify" — a smart push notification 
platform for mobile apps. Modern dark theme, teal accent color. Include: sticky navbar 
with logo and CTA, hero section with headline and subheadline, 3 feature cards, 
pricing table with 3 tiers, testimonials section, and footer with newsletter signup.
```

**What gets created:**
```
./notify-landing/
├── index.html    (full semantic HTML structure)
├── style.css     (dark theme, teal accents, responsive)
└── script.js     (smooth scroll, mobile menu toggle, form validation)
```

**Key features generated:**
- Sticky navigation with mobile hamburger menu
- Hero with gradient text headline
- CSS Grid feature cards with hover effects
- Pricing cards with "Most Popular" badge
- Testimonial carousel
- Newsletter form with validation
- Responsive breakpoints for tablet and mobile

## Opening Your Generated Website

After Vega generates files, open them immediately:

```bash
# macOS
open ./notify-landing/index.html

# Linux
xdg-open ./notify-landing/index.html

# Windows
start ./notify-landing/index.html
```

Or use a simple local server for a more realistic preview:

```bash
# Python (built-in)
cd notify-landing && python -m http.server 3000

# Node.js (npx)
cd notify-landing && npx serve .
```

Then open `http://localhost:3000` in your browser.

## Refining Your Website in Session

You don't need to get everything perfect in one prompt. Iterate:

```
You: Change the hero section background to a dark gradient from #0a0a0a to #1a1a2e

You: Add a "Trusted by 500+ teams" social proof line under the hero CTA

You: Make the pricing cards animate in with a stagger effect on scroll

You: Add a cookie consent banner at the bottom of the page
```

Each follow-up updates only what you ask — Vega preserves the rest.
