# Models

Vega supports 15+ AI models across 4 providers. You can use free-tier models to get started at no cost, and switch between providers and models at any time.

---

## Default Model

Vega starts with **`meta/llama-3.1-405b-instruct`** (NVIDIA NIM) as the default model. It offers an excellent balance of:
- Code quality and reasoning depth
- Long context window (128K tokens)
- Free-tier availability via NVIDIA NIM

The first-run wizard lets you choose your preferred provider and sets the appropriate default model for that provider.

## How to Switch Models

### Option 1: Interactive picker (`/switch`)

The easiest way — shows the 5 primary curated models:

```
  ✦ Switch Active Model

    1.   Llama 3.1 405B (NVIDIA) (meta/llama-3.1-405b-instruct)
    2.   Gemini 2.0 Flash (Google) (gemini-2.0-flash)
    3. ★ Llama 3.3 70B (Groq) (llama-3.3-70b-versatile)        ← active
    4.   DeepSeek-V3 (DeepSeek) (deepseek-chat)
    5.   Kimi K2.6 (NVIDIA) (moonshotai/kimi-k2-instruct)

  Choose a model [1-5]:
```

### Option 2: Set model by ID (`/model`)

```bash
# View current model
/model

# Set a specific model
/model deepseek-chat
/model gemini-2.0-flash
/model moonshotai/kimi-k2-instruct
```

### Option 3: Switch provider (`/provider`)

Switches to a provider and automatically uses that provider's default model:

```bash
/provider google     # → switches to gemini-2.0-flash
/provider groq       # → switches to llama-3.3-70b-versatile
/provider deepseek   # → switches to deepseek-chat
/provider nvidia     # → switches to meta/llama-3.1-405b-instruct
```

### Option 4: Start Vega with a specific model

```bash
vega --model deepseek-chat
vega --model gemini-2.0-flash --provider google
```

### Option 5: List all models

```bash
# Inside Vega
/models

# Or as a shell command
vega models

# Filter by provider
vega models nvidia
vega models groq
```

## Which Model Should I Use?

### For most users — `meta/llama-3.1-405b-instruct` (NVIDIA, Free)

The Vega default. Excellent code quality, 128K context, and available for free on NVIDIA NIM.

### For code generation & debugging — `moonshotai/kimi-k2-instruct` (NVIDIA, Free)

Kimi K2 is Vega's CodeAgent and DebugAgent choice. Outstanding at generating multi-file projects and complex debugging scenarios.

### For maximum speed — `meta-llama/llama-4-scout-17b-16e-instruct` (Groq, Free)

Groq's hardware provides dramatically faster inference — responses in under 1 second. Use this when you want near-instant output for Q&A and explanations.

### For architecture & planning — `deepseek-chat` (DeepSeek, Cheap)

DeepSeek-V3 excels at structured thinking, system design, and code review. Very cost-effective at approximately $0.27/million input tokens.

### For long context — `gemini-1.5-pro` (Google, Free)

2M token context window — use this when working with large codebases, multiple files, or very long documents.

## Model Persistence

When you switch models, Vega saves the selection to `~/.vega/config.json` so it persists across sessions:

```bash
/config model
# → model = moonshotai/kimi-k2-instruct
```

To reset to the default:

```bash
/reset
```
