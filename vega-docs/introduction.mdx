# What is Vega?

**Vega** is an open-source, AI-powered command-line interface built by [Raimic Labs](https://github.com/Raimic-Labs). It turns your terminal into an intelligent coding partner — one that understands your intent, picks the right AI agent, and produces working code directly in your project directory.

> **Code at the speed of stars.** ✦

Instead of copy-pasting from a browser or switching between chat tabs, you simply open your terminal, run `vega`, describe what you want to build, and watch it happen.

## Key Features

- 🤖 **6 Specialized AI Agents** — each optimized for a different domain (code, image, fast Q&A, planning, debugging, review)
- 🌐 **Multi-Model Support** — NVIDIA Kimi K2, Llama 3.1 405B, Gemini 2.0 Flash, Groq Llama 4, DeepSeek-V3, and more
- ⚡ **Instant Project Generation** — full websites, REST APIs, React apps, Python scripts from one prompt
- 🔑 **Flexible API Keys** — use free-tier keys from NVIDIA, Gemini, Groq, DeepSeek — no paid subscription required
- 💬 **Conversational Context** — Vega remembers your session so you can refine and iterate naturally
- 🛠️ **Slash Commands** — built-in shortcuts like `/connect`, `/switch`, `/build`, `/agents` for power users
- 📁 **Direct File Output** — generated code is written to your working directory, ready to run
- 🔄 **Live Model Switching** — swap models or providers mid-conversation with `/switch`
- 🤖 **Autonomous ReAct Agent** — `/agent <goal>` runs a multi-step reasoning loop with tool use

## How It Works

1. **You type a prompt** — natural language, plain English, as specific or vague as you want
2. **Vega routes to the right agent** — it analyzes your intent and picks the best specialized agent
3. **The agent generates code** — using the configured AI model, it produces complete, working files
4. **Files land in your directory** — no clipboard needed; Vega writes directly to your filesystem
5. **Iterate naturally** — follow up, refine, ask questions — Vega keeps the context throughout
